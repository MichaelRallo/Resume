Latin Square Puzzle (AKA MineSweeper Queens)

Description

This Latin Square Puzzle was comprised of a grid with numbers and shaded cells. Given a grid with numbers, I developed an algorithm to fill in the shaded areas whilst meeting speicifc guidelines. These guidelines are that each number must contain shaded squares within 1 space of its vacinity and that the number of adjacent squares is depicted by the number on the grid itself. No shaded square can be in the same row or column. For the search/ai algorithms, I used BackTracking Search with Minimum Remaining Value (MRV), using degree heuristics and forward checking.

Features

  • -Made using JavaFX
  • -Can solve any possible Grids
  • -BackTracking Search
  • -Forward Checking
  • -MRV and Hueristic Values

Virtual CPU

Description

This Project was carried out as part of my Operating Systems course at the University of Missouri - Columbia. It is an application that simulates how a processes handles processes from a ready Queue using specified algorithms. The user can choose between using Round Robin Scheduling or FCFS scheduling. Depending on how many of each algorithm is called, threads will be created to handle processing the Queue. In essence, a file full of PCB data is read in and threads are created to processes a single dynamic array made up of PCB input based on the desired algorithm.

Features

  • -Utilizes Dynamic Linked Lists
  • -Uses System Calls for faster performace.
  • -Demonstrates how C's pthreads work.
  • -Can perform 'infinite' series of algorithms.
  • -Can Track How the cpu handles different algorithms and Scheduling techniques.
  • -Uses CMake to build/compile related libraries and source code.

Source Code: Github-VCPU
